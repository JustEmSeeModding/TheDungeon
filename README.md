# The Dungeon - Minecraft NeoForge Mod

---------
A comprehensive Minecraft mod built with NeoForge that adds a dungeon-themed experience to Minecraft 1.21.1, featuring new items, entities, mechanics, and progression systems.

## Features

### Items & Equipment
- Custom weapons and tools with unique properties
- Specialized armor sets with different materials
- Unique food items with special effects

### Entities & Mobs
- Custom mob entities with unique behaviors
- Entity-specific mechanics and AI

### Dungeon Class System
- Multiple dungeon classes for character progression
- Subclass specializations for varied playstyles
- Class-based item and equipment restrictions
- Custom attributes tied to class selection

### Gameplay Mechanics
- Unique recipes and crafting systems
- Particles and visual effects

### Tools & Integration
- JustEnoughItems (JEI) support for recipe browsing
- Patchouli guide book integration
- Curios API support for accessory items
- Advanced item tooltip system

## Requirements

- **Minecraft Version**: 1.21.1
- **Modding Framework**: NeoForge 21.1.191 or later

## Dependencies

- **JEI (Just Enough Items)**: For recipe integration
- **Patchouli**: For in-game guide books
- **Curios API**: For accessory/curio item support
